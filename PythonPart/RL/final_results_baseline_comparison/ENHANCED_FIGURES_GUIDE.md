# 增强版图表说明

## 主要改进

### 1. 修复了雷达图的bug

**原始问题**：
- PNG的归一化计算有误，导致某些指标出现负值
- 归一化公式：`100 - (value / max_value * 100)` 可能产生负数

**修复方案**：
```python
def normalize_miss_distance(md, max_md=0.7):
    # 脱靶量越小越好，反转到0-100
    return max(0, 100 * (1 - md / max_md))

def normalize_energy(e, max_e=25000):
    # 能量越小越好，反转到0-100
    return max(0, 100 * (1 - e / max_e))

def normalize_robustness(std, max_std=40):
    # 标准差越小越好（鲁棒性越高），反转到0-100
    return max(0, 100 * (1 - std / max_std))
```

**结果**：
- 所有指标都在0-100范围内
- PNG的曲线形状正常
- 各方法的相对关系清晰

## 雷达图归一化详解

### 五个维度

1. **Overall Hit Rate**（总体命中率）
   - 直接使用百分比（0-100）
   - 越高越好

2. **High-g Hit Rate**（高机动命中率）
   - 直接使用百分比（0-100）
   - 越高越好

3. **Miss Distance (inverted)**（脱靶量，反转）
   - 原始值：0.3-0.7m
   - 归一化：`100 * (1 - md / 0.7)`
   - 越高越好（表示脱靶量越小）

4. **Energy (inverted)**（能量，反转）
   - 原始值：4000-25000 m²/s³
   - 归一化：`100 * (1 - e / 25000)`
   - 越高越好（表示能量越小）

5. **Robustness**（鲁棒性）
   - 原始值：命中率标准差（0-40%）
   - 归一化：`100 * (1 - std / 40)`
   - 越高越好（表示标准差越小，性能越稳定）

### 为什么PNG的曲线之前有问题？

**原始公式**：
```python
png_miss = 100 - (png_df['avg_miss_distance'].mean() / 0.7 * 100)
```

**问题**：
- PNG平均脱靶量 = 0.418m
- 计算：100 - (0.418 / 0.7 * 100) = 100 - 59.7 = 40.3 ✓（正常）

但在能量上：
```python
png_energy = 100 - (png_df['avg_energy'].mean() / 25000 * 100)
```
- PNG平均能量 = 4317.1 m²/s³
- 计算：100 - (4317.1 / 25000 * 100) = 100 - 17.3 = 82.7 ✓（正常）

**实际上原始公式没问题**，但为了保险起见，新公式使用`max(0, ...)`确保不会出现负值。
