# 论文综述与中期答辩准备分析

## 一、六篇论文核心梳理

### 1. He et al. — Computational Missile Guidance: A Deep RL Approach (AIAA, 2021)
- **场景**：2D平面末制导，质点模型，弹目匀速假设
- **算法**：DDPG（深度确定性策略梯度），输出横向加速度指令
- **状态空间**：视线角λ、弹目相对距离r、径向/横向相对速度Vr/Vλ、弹体航迹角γM、当前加速度aM
- **奖励设计**：四项启发式奖励 = 控制能量惩罚(ra) + ZEM惩罚(rz) + 逼近奖励(rVr) + 命中奖励(rr)
- **自动驾驶仪**：一阶惯性环节 τ·ȧM + aM = ac
- **关键结论**：(1) 带先验知识学习（固定PN结构学增益）优于从零学习；(2) DDPG制导律精度高于PNG，对不同场景有泛化能力
- **与你的相关性**：★★★★★ 最直接的基线参考——2D质点+DDPG+简单奖励设计，可作为你第一版实现的模板

### 2. Hong et al. — Study on RL-Based Missile Guidance Law (Appl. Sci., 2020)
- **场景**：2D平面，反舰导弹拦截船只，稀疏奖励环境
- **算法**：改进DDPG，每步奖励设为该episode最终得分（缓解稀疏奖励）
- **状态空间**：仅用LOS角速率Ω和导弹速度Vm（与PNG输入完全相同，保证公平对比）
- **控制器模型**：一阶惯性延迟 τ·ȧm + am = ac
- **关键结论**：(1) RL制导律在理想环境下约等于PNG（PNG本身准最优）；(2) 当引入控制器延迟后，RL鲁棒性显著优于PNG；(3) 公平对比要求输入维度一致
- **与你的相关性**：★★★★★ 最简洁的实现范本——输入仅需Ω和Vm，天然适配红外导引头（无距离信息）的约束

### 3. Li et al. — Missile Guidance with Assisted DRL for Head-on Interception (CIS, 2022)
- **场景**：3D空间，迎头拦截9g机动目标，含观测噪声与延迟
- **算法**：ARL = PPO + 辅助监督学习（预测目标加速度） + 高斯自模仿学习(GSIL)
- **状态空间**：弹目相对距离D、接近速度Ḋ、高低角/方位角及其速率、导弹速度/弹道角
- **奖励设计**：即时奖励（距离缩减+迎头角余弦）+ 终端奖励（命中且大迎头角）
- **关键结论**：(1) 辅助学习显著提升噪声环境下训练效率；(2) GSIL改善连续动作空间探索；(3) 在噪声+延迟+机动目标综合条件下优于传统方法
- **与你的相关性**：★★★★☆ 提供了3D扩展路径和处理噪声/延迟的实用技术（辅助任务思路）

### 4. Yan et al. — A Hierarchical RL Method for Missile Evasion and Guidance (Sci. Rep., 2022)
- **场景**：2D平面，TMD问题（攻击弹需同时制导打目标+规避拦截弹）
- **算法**：分层PPO——底层两个智能体（制导+规避），高层选择器决定每时刻激活哪个
- **状态空间**：制导智能体用(dm, λm, ḋm, λ̇m)；规避智能体用(di, λi, ḋi, λ̇i)
- **奖励设计**：分层独立奖励——制导关注ZEM/能耗/飞行时间/FOV约束；规避关注不被截+不丢目标
- **关键结论**：(1) 单一PPO无法完成制导+规避复合任务；(2) 分层结构100%成功率；(3) 对自驾仪二阶延迟和测量噪声鲁棒
- **与你的相关性**：★★★☆☆ 为后续扩展提供"攻防故事"框架（导师要求考虑全程攻防），但初期不必实现

### 5. Gaudet & Furfaro — Integrated and Adaptive G&C via RL Meta-Learning (2023)
- **场景**：完整6-DOF，空对空导弹，十字舵控制，含径向天线罩效应和柔性体动力学
- **算法**：Meta-RL（PPO+LSTM隐状态实现在线自适应），输出舵面偏转速率
- **状态空间**：导引头输出（LOS角速率替代量）、陀螺角速率、加速度计、舵面偏转角
- **制导控制一体化**：策略网络直接替代传统"导引律+三回路自驾仪"的全部功能
- **关键结论**：(1) 一体化策略大幅超越传统PN+三回路基准；(2) 不需要测高度/速度；(3) LSTM是关键——去掉循环层性能崩溃；(4) 红外导引头（无距离/距离率）情况仍可工作，精度虽降但仍优于传统基准
- **与你的相关性**：★★★★★ 这是你课题最终目标的参考范本——完整6DOF+IGC一体化+红外导引头。但复杂度极高，应作为远期对标而非第一步实现

### 6. Jeng & Chiang — End-to-End Autonomous Navigation via DRL (Sensors, 2023)
- **场景**：2D平面，轮式机器人自主导航避障
- **算法**：DDPG和TD3对比，提出生存惩罚函数解决稀疏奖励
- **关键结论**：(1) TD3在收敛速度和稳定性上优于DDPG；(2) 连续episode连接策略有效防止训练失败
- **与你的相关性**：★★☆☆☆ 非导弹领域，但提供DRL方法论参考（TD3优势、奖励塑造思路）

## 二、痛点分析（对应导师三大痛点）

### 痛点1：复杂目标（高机动、具备AI）

**传统方法的本质局限：**
- PNG系列制导律基于线性化运动学推导，其最优性仅在目标非机动或低机动时成立（He论文、Hong论文均明确指出）
- 当目标执行高g机动（≥6g），传统PN+三回路系统命中率急剧下降——Gaudet论文的基准测试显示，8g目标下100cm以内命中率仅25%
- 传统方法需要估计剩余飞行时间（tgo）或目标加速度，而高机动目标使这些估计极不可靠（Li论文强调）

**DRL的突破方向：**
- 从数据中学习非线性映射，无需tgo或目标加速度的显式估计
- Li论文用辅助监督任务隐式学习目标加速度模式，将提取到的信息融入策略
- Yan论文的分层架构应对"目标同时具备智能规避能力"的情况

### 痛点2：强对抗环境——鲁棒性需求

**传统方法的脆弱性：**
- PID/增益调度高度依赖线性化工作点，偏离设计包线后性能恶化
- 固定增益无法适应飞行包线内气动参数的大范围变化
- 天线罩折射+导引头不完美稳定+陀螺偏差→寄生姿态回路→传统系统极易不稳定（Gaudet论文详述）

**DRL的鲁棒性来源：**
- 训练时通过域随机化（Domain Randomization）暴露多种扰动：气动系数偏差、传感器噪声、执行器延迟
- Hong论文核心发现：RL制导律在加入控制器延迟后性能衰减远小于PNG
- Gaudet论文训练时随机扰动气动系数±10%，部署时即使遇到±20%~30%偏差或未建模柔性体仍保持性能
- LSTM的在线自适应：无需重新训练，隐状态h实时演化应对环境变化

### 痛点3：全态势信息（体系化作战）

**传统方法的信息瓶颈：**
- PNG仅使用LOS角速率，最多加接近速度——信息维度极低
- 三回路自驾仪各通道独立设计，无法利用跨通道耦合信息
- 面对体系化作战中多源异构信息（友机态势、威胁告警、指挥链指令），传统方法无接入接口

**DRL的扩展能力：**
- 神经网络天然支持高维输入，Li论文已使用10维观测向量
- Gaudet论文一体化策略直接处理导引头+陀螺+加速度计的多传感器融合输出
- 分层架构（Yan论文）支持上层接入态势信息，下层执行制导控制——这就是导师说的"全程攻防故事"的架构雏形

## 三、需求总结——"小导弹打无人机"场景约束

根据导师指导，你的场景是：**红外导引头微型防空导弹拦截小型无人机**。这定义了如下约束：

| 约束项 | 具体含义 | 对实现的影响 |
|--------|---------|-------------|
| 红外导引头 | 仅能测量目标角度（方位角、高低角），**无距离、无距离率** | 状态空间不能含r和ṙ——参考Gaudet论文IV.D节"无距离"实验 |
| 微型弹体 | 低雷诺数、气动性能有限、控制力矩小 | 最大可用过载受限，控制带宽低 |
| 目标=无人机 | 速度相对较低（~50-150m/s），但机动性极强（可能>10g瞬时） | 速度优势大但角度追踪压力大 |
| 简单导引头 | 硬件成本低、视场角有限 | 需考虑FOV约束，制导律必须保持目标在视场内 |
| 全程制导 | 初/中/末三段都要考虑 | 初中段可用惯导+指令修正，末段切换红外自主寻的 |

## 四、简化任务方案——三步走路线

### 第一步：2D质点模型 + 比例导引基线（1-2周）

**目标**：建立最简可运行的仿真闭环，产出传统基线数据。

**具体内容**：
1. **运动学模型**：参考He论文和Hong论文的2D相对运动方程
   - 弹目相对距离 ṙ = VT cos(γT-λ) - VM cos(γM-λ)
   - LOS角速率 r·λ̇ = VT sin(γT-λ) - VM sin(γM-λ)
   - 航迹角变化 γ̇ = a/V
2. **PNG制导律**：ac = N·VM·Ω（N取3~5）
   - 注意：红外导引头无距离项——先用真值实现，后续再退化为仅角度信息
3. **自动驾驶仪**：一阶惯性模型 τ·ȧm + am = ac（τ=0.1~0.5s）
4. **目标模型**：无人机匀速飞行 + 随机方向的阶跃/正弦机动（最大5~15g）
5. **终止条件**：弹目距离<阈值 或 超时 或 目标出视场
6. **评价指标**：ZEM*脱靶量、飞行时间、能量消耗∫a²dt

**输出**：Python脚本可运行，产出PNG在不同初始条件/目标机动下的脱靶量统计表。

### 第二步：DRL制导律训练与对比（2-3周）

**目标**：用强化学习替代PNG，完成A/B对比。

**具体内容**：
1. **环境封装**：将第一步的仿真包装为Gymnasium环境
   - 状态（方案A，仅角度）：[λ̇, VM, aM/amax]（对标Hong论文的公平输入）
   - 状态（方案B，含距离）：[r, Vr, Vλ, γM, aM]（对标He论文）
   - 动作：归一化加速度指令 a ∈ [-1, 1]，映射到[-amax, amax]
2. **奖励函数**（参考He论文四项结构）：
   - r = -kz·(ZEM/ZEM0)² - ka·(aM/amax)² + kVr·I(Vr>0) + kr·I(命中)
3. **算法选择**：
   - PPO（推荐起手）：训练稳定，适合首次尝试——Yan和Li都用PPO
   - DDPG/TD3：He和Hong使用，可作为对比
   - 库：Stable-Baselines3（Python生态完善）
4. **训练配置**：
   - Episode：每次随机初始化弹目几何关系
   - 目标机动：随机bang-bang或正弦规避
   - 域随机化：扰动τ（自驾仪延迟）、VM（弹速）、噪声

**输出**：RL制导律 vs PNG的对比数据表——脱靶量均值/标准差、命中率、能耗，分有/无控制器延迟两种工况。

### 第三步：扩展为3D + 红外导引头约束（2-3周）

**目标**：逼近真实场景，完善中期答辩内容。

**具体内容**：
1. **3D运动学**：参考Li论文的三维弹目模型（高低角+方位角）
2. **红外导引头模拟**：
   - 仅输出方位角qb和高低角qe及其速率
   - 无距离/距离率
   - 加高斯噪声（σ ~ 0.5-1 mrad）
   - FOV约束（视场角±30°~45°）
3. **状态空间退化**：去掉距离项，参考Gaudet论文IV.D节
4. **攻防故事框架**（初步）：
   - 初段：惯导飞向目标预测区域
   - 中段：指令修正（假设数据链提供目标大致位置）
   - 末段：红外锁定后自主寻的（RL制导律接管）

**输出**：3D仿真结果 + 红外约束下的性能对比 + 全程制导流程图。

## 六、关键公式速查

**比例导引律**：ac = N · VM · λ̇

**2D相对运动学**：
- ṙ = VT cos(γT - λ) - VM cos(γM - λ)
- r·λ̇ = VT sin(γT - λ) - VM sin(γM - λ)

**零控脱靶量ZEM**：z = r·Vλ / √(Vr² + Vλ²)

**一阶自驾仪模型**：τ·ȧM + aM = ac

**二阶自驾仪模型**（Yan论文）：ä = -2ξωn·ȧ - ωn²·a + ωn²·ac

**航迹角变化率**：γ̇ = a / V
